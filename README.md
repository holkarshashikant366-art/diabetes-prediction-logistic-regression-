# Diabetes Prediction using Logistic Regression

## Overview
This project predicts whether a patient has diabetes using Logistic Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Workflow
1. Data Preprocessing
2. Feature Scaling using StandardScaler
3. Train-Test Split
4. Logistic Regression Model Training
5. Model Evaluation

## Results
- Accuracy: 90%

## Key Learning
Feature scaling improved model performance and helped Logistic Regression achieve better classification accuracy.
